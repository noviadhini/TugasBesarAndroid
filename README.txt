Aplikasi Visit Malang merupakan aplikasi sistem informasi malang raya berbasis Android, 
bukan hanya sebagai temapt informasi tetapi aplikasi ini juga dilengkapi dengan fitur google map sehingga
memudahkan user dalam mencari lokasi wisata, lokasi pendidikan, lokasi kuliner, lokasi penginapan.

Aplikasi Visit Malang dibagi menjadi 2 pengguna :
1. Untuk Admin : Dioperasikan di Web Visit Malang , Admin mengiputkan data informasi wisata, pendidikan, kuliner, penginapan.
2. Untuk User  : Dioperasikan di Android, User dapat melihat semua data informasi malang raya.
